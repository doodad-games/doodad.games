Privacy Policy

Games

Some of our games may collect user data to deliver personalised ads, or to allow us to understand how users interact with our games so we can best improve them.

Collection of data is performed via Unity Ads and Unity Analytics. Please read the Unity privacy policy to learn more about how they collect and manage your data, and instructions to opt-out of services or data collection where appropriate.

Information regarding the data collected for each of our games is listed below.

FrankenStorm; Lonely Birds; Neanderfall; Space Trash; uniTower; Viking Trickshot; Villager

Only standard Unity Analytics data is collected. (No ads)

Bombardier

  • Standard Unity Analytics data. (Ads are not served)
  • Information when the following in-game events occur:
    • When a new game is started.
    • When a game has ended, along with the following metadata:
      • Game duration.
      • Score.
      • Cause of the game's end.

Other

This website uses no tracking or analytics, so no data is collected.
